# Recommended Tools

Use this page for tools that repeatedly save time or reduce project risk.

## Starter List

- Version control: Git and GitHub
- Package management: Bun, npm, pnpm
- UI inspection: browser DevTools and Playwright screenshots
- API testing: curl, Bruno, or Postman
