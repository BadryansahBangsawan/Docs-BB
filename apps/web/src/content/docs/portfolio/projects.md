## Projects

### Nontonkuy

![Nontonkuy screenshot](https://bwwpjdcrelzvdzdcioop.supabase.co/storage/v1/object/public/projects/admin-projects/1772613857936-screenshot-2026-03-04-at-16.34.39-1.webp "Nontonkuy")

Full-stack content platform yang dibangun sebagai monorepo dengan Bun dan Turborepo. Frontend memakai Next.js App Router, sementara autentikasi, sesi, dan integrasi storage dipisah ke Cloudflare Worker, D1, dan R2. Project ini juga memakai proxy API internal supaya integrasi konten eksternal dapat dikontrol dari sisi server.

- Stack: Full Stack Next.js, TypeScript, Bun, Turborepo, Cloudflare Worker, D1, R2
- Focus: platform konten, autentikasi terpisah, integrasi provider eksternal
- Links: [Live Demo](https://nontonkuy.badry.asia/), [Source Code](https://github.com/BadryansahBangsawan/nontonkuy)
