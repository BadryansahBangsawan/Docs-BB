# Tamplateku

![Tamplateku screenshot](https://bwwpjdcrelzvdzdcioop.supabase.co/storage/v1/object/public/projects/admin-projects/1776941458227-screenshot-2026-04-23-at-18.50.12.webp "Tamplateku")

Tamplateku adalah website layanan pembuatan website profesional. Halaman ini diarahkan untuk menjelaskan penawaran layanan dengan cepat, menampilkan konteks bisnis, dan memberi jalur aksi yang jelas untuk calon klien.

Project ini memakai TanStack, Express, Tailwind, dan GSAP. Kombinasi tersebut cocok untuk service website yang membutuhkan routing modern, backend ringan, styling cepat, dan interaksi visual yang lebih hidup.

## Stack

- Tanstack
- Express
- Tailwind
- Gsap

## Notes

The source content describes the implementation as a website creation service using TanStack.

## Detail

- Fokus utama halaman adalah menjelaskan layanan pembuatan website secara langsung.
- Tailwind dipakai untuk membangun UI yang konsisten dan mudah diubah.
- Express memberi ruang untuk kebutuhan server/API ringan di luar sisi frontend.
- GSAP memberi dukungan animasi untuk meningkatkan kesan visual tanpa mengubah inti konten layanan.
