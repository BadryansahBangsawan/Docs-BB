# Mts Darussalam

![Mts Darussalam screenshot](https://bwwpjdcrelzvdzdcioop.supabase.co/storage/v1/object/public/projects/mts-darussalam.png "Mts Darussalam")

Mts Darussalam adalah website profil sekolah yang berfokus pada informasi institusi dan publikasi pendidikan. Project ini membantu sekolah memiliki kanal digital yang dapat memperkenalkan profil, kegiatan, dan informasi penting kepada calon siswa, wali, serta masyarakat.

Untuk website sekolah, konten perlu mudah dipahami dan tidak terlalu ramai. Informasi utama harus tersusun rapi agar pengunjung dapat menemukan identitas sekolah, publikasi, dan informasi pendidikan dengan cepat.

## Stack

- Next.js
- TypeScript
- TailwindCSS

## Category

Education

## Detail

- Website diposisikan sebagai profil dan media informasi sekolah.
- Struktur halaman mendukung publikasi pendidikan dan pengenalan institusi.
- Stack Next.js, TypeScript, dan TailwindCSS cocok untuk website profil yang ringan dan mudah dipelihara.
