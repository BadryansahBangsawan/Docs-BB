# LKPS

![LKPS screenshot](https://bwwpjdcrelzvdzdcioop.supabase.co/storage/v1/object/public/projects/lkps.png "LKPS")

LKPS adalah platform website untuk kebutuhan edukasi dan pelatihan. Project ini menampilkan informasi program, kredibilitas lembaga, dan jalur akses bagi pengguna yang mencari layanan pelatihan.

Karena masuk kategori education/training, konten harus membantu pengguna memahami nilai program dengan cepat. Halaman perlu menjelaskan konteks lembaga, layanan yang tersedia, dan cara pengguna melanjutkan tindakan berikutnya.

## Stack

- Next.js
- TypeScript
- TailwindCSS

## Category

Education/Training

## Detail

- Fokus utama adalah penyajian informasi edukasi dan pelatihan.
- Konten perlu menonjolkan kredibilitas, program, dan akses layanan.
- Next.js, TypeScript, dan TailwindCSS dipakai untuk menjaga tampilan tetap rapi dan responsif.
