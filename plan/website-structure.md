# Personal Docs - Deep Website Structure (Template)

This is a generic template structure based on a documentation site pattern.

## 1. Main Categories & Landing Pages
- `/` (Root/Home)
- `/ai-llm` (AI & LLM Topics)
- `/portfolio` (Client Work/Case Studies)
- `/project` (Personal & Professional Projects)
- `/useful-link` (Resources & Links)

## 2. AI & LLM
- `/ai-llm/concept-a` (e.g., prompt-engineering)
- `/ai-llm/concept-b` (e.g., local-models)
- `/ai-llm/concept-c`

## 3. Portfolio (Client / Agency Work)
- `/portfolio/client-a/project-1`
- `/portfolio/client-a/project-2`
- `/portfolio/client-b/project-1`

## 4. Projects By Category

### Company / Corporate
- `/project/company/company-a`
- `/project/company/company-b`
- `/project/company/company-c`

### News / Content Portals
- `/project/news/news-portal-a`
- `/project/news/news-portal-b`

### SaaS (Software as a Service)
- `/project/saas/saas-app-a`
- `/project/saas/saas-app-b`
- `/project/saas/saas-app-c`

### Organization / Personal
- `/project/organization/org-name`
- `/project/personal/my-side-project`

## 5. Useful Links
- `/useful-link/resource-1` (e.g., recommended-tools)
- `/useful-link/resource-2` (e.g., reading-list)
